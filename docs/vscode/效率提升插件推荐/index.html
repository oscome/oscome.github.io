    本期给大家推荐几个通用型的插件，无关乎语言层面，希望能提升你的工作效率，增进你的视觉体验。

    vscode-icons #

    地址： https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons

    这个插件会根据文件后缀、名称显示不同的 icons，颜值颇高，设置后看着各种小图标心情舒畅，颜值党值得一试，如下图。（也可以自己设置喜欢的图标主题，扩展栏搜索 tag:icon-theme 安装你喜欢的即可）

    Code Runner #

    地址: https://marketplace.visualstudio.com/items?itemName=formulahendry.code-runner

    这是小欧近期学习C++时发现的，可运行多种语言的代码片段，如 C, C++, Java, JavaScript, PHP, Python,, Go, Lua, Groovy, PowerShell, BAT/CMD, BASH/SH,, Visual Basic .NET, Clojure, Haxe, Objective-C, Rust, Racket, Scheme, AutoHotkey, AutoIt, Kotlin, Dart……

    使用方法非常简单，光标在需要 debug 的文件中，然后 control+option+N，就可以 run code 啦！效果如图：（PS：你也可以自行定义命令。没错， –std=c++11 就是小欧自定义的）

    Project Manager #

    地址： https://marketplace.visualstudio.com/items?itemName=alefragnani.project-manager

    不知道大家在实际工作中，是否像小欧一样需要在多个项目间反复横跳。如果是这样，并觉得 VSCode 的下方历史项目操作起来略显繁琐，那么 project manager 绝对值得一试。

    安装完成后，左侧会有快捷选项卡，方便！！！可以点击上方修改图标自行配置，比如我的:​

    Better Comments #

    地址： https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments

    又是颜值党的福音，美化注释直选，可以通过第一个字符区分不同的注释，当然也可以自定义，@和diy都是小欧自定义的，不同颜色也更方便区分不同级别的注释，好看又实用。

    TODO Tree #

    地址： https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree

    TODO Tree 可以很方便地显示当前项目中所有的TODO所在的文件及位置，注意是大写的TODO哦，效果如图：（左侧也是有个选项卡快捷入口的）

    Error Lens #

    地址： https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens

    ErrorLens 通过使诊断更加突出来增强语言诊断功能，在语言生成诊断的任何位置突出显示整行，并内联打印消息，效果如图。（PS：不得不说，可能对各种语言的支持并没有那么强大，有些提示并不精准，但无伤大雅）
