本期给大家推荐思维导图软件，一般也都支持流程图、组织架构图、UML等。

drawio

项目地址: https://app.diagrams.net/

drawio 之前在推荐 VSCode 扩展曾提到过，免费且好用。

第一次打开网站比较慢，因为国外资源且加载较多资源，稍稍等一会就好，用多了就没问题了，支持多种存储。

ProcessOn

项目地址：https://www.processon.com/

ProcessOn 可用于原型图、UML、BPMN、网络拓扑图等多种图形绘制，另外还可以进行团队协同。但不开通会员注册后能文件最多可以保存 14 个（现在好像是9个），如果你使用频率不高（一月一两次），用这个绰绰有余啦，毕竟可以导入导出，导入的格式也支持很多，xmind、mmap 甚至是 excel。

另外，ProcessOn 有了移动版，这个是值得称赞的。如果你受限于文件限制，可以开通会员，一般有优惠码大概100出头，年付！

值得一提的是，你可以把好的图制作成模板，设置成付费。

XMind

项目地址： https://xmind.cn/

Xmind 应该是大家熟知的，毕竟做了十多年了，也足够好用。即使试用版本，也能应付一般的场景。导入支持 word、markdown等，同样支持团队协作，另外有教育特别优惠。

百度脑图

项目地址： https://naotu.baidu.com/

印象中至少在六年前就有了，现在来看 UI 有点儿粗糙，但能用且不收费。

亿图

项目地址: https://www.edrawsoft.cn/

不知道是记忆问题还是，感觉跟之前的界面大不一样了，一般使用场景试用版也够用了。

总结

如果不想花钱，建议使用 drawio、Xmind ，如果不介意收费，ProcessOn 也是不错的选择，如果想赚钱，ProcessOn 社区，这个还是有难度的。

你在使用什么工具画流程图、思维导图呢，不妨分享给大家？

关注和赞赏都是对小欧莫大的支持！ 🤝 🤝 🤝
